About The Project

Intelligent AI Solution

A cutting-edge chatbot built on IBM Cloud infrastructure, combining Watson Assistant's natural language processing with robust data management to deliver instant, accurate responses to student queries.

This project revolutionizes student support by leveraging IBM Watson's advanced AI capabilities. Students can instantly access academic resources, exam schedules, fee information, and faculty contacts through natural conversation.

Built with scalability and reliability in mind, the system integrates Cloudant DB for efficient data management and Cloud Object Storage for seamless document retrieval, ensuring students get the information they need, whenever they need it.
